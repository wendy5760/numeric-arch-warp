# GitHub Publishing Notes

## Suggested repository description

Figma plugin for numeric raster arch warping of frames and images.

## Suggested repository topics

figma-plugin, figma, image-warp, raster, design-tools, plugin, canvas

## Suggested first release title

Numeric Arch Warp v0.1.0

## Suggested first release notes

Initial public version of Numeric Arch Warp.

Included:

- Numeric arch warp controls for selected frames and images.
- Upward and downward bend amounts.
- Adjustable curve shape and sampling segment count.
- Saved settings on generated warp results.
- Replacement workflow for applying the same warp to new artwork.
- Automatic safe raster fitting to avoid Figma image-size errors.
- No external network access.

Known limitation:

- Warp outputs are transparent PNG results. Original source frames remain editable separately.
