# Numeric Arch Warp

这是一个 Figma 开发插件原型，用来把选中的 Frame、图片、组件实例或普通图层整体做“向上/向下拱起”的数值化弯曲。

它和手动拖锚点的思路不同：弯曲由明确参数控制，所以同一组数值可以复用到多张替换图片上。

## 使用方式

1. 在 Figma 中打开 `Plugins > Development > Import plugin from manifest...`。
2. 选择本文件夹里的 `manifest.json`。
3. 选中一个 Frame 或图片，运行 `Numeric Arch Warp`。
4. 点击“载入当前选择”，调整弯曲强度、曲线形状和采样段数。
5. 点击“生成副本”。

如果希望 Frame 和图片一起弯曲，请选中外层 Frame。插件会先把整个 Frame 导出成一张源图，再把源图内容重新采样成拱起效果，所以边框、阴影、图片内容会一起变形。

## 替换图片并保持同样弯曲

1. 选中新图片或新 Frame。
2. 同时选中之前生成的弯曲结果。
3. 点击“载入当前选择”。
4. 点击“更新已选结果”。

插件会从旧的弯曲结果里读取保存的参数，然后把同一组弯曲应用到新的源图上。

## 参数

- `弯曲强度 (%)`：正数向上拱起，负数向下拱起。
- `曲线形状`：默认 `2` 是抛物线。数值越大，中间越平、两侧变化越集中。
- `采样段数`：越高越平滑，图片越大时可以提高到 `512` 或 `1024`。
- `导出倍率`：源图导出精度。常规使用 `2x`，大图或最终稿可用 `3x` / `4x`。如果图片或 Frame 过大，插件会自动降到 Figma 可接受的安全像素尺寸。

## 重要限制

Figma 插件 API 不能创建一个真正“实时弯曲、内部图层仍完全可编辑”的原生 Frame。这个插件生成的是一个带透明像素的 PNG 结果，并把弯曲参数写入该结果的 plugin data。

也就是说：源 Frame 仍然可以保留和编辑；需要替换图片时，再运行插件更新一次结果。这样才能得到和 Warp Drive 类似的内容级变形，同时又能保持参数一致。

如果界面里出现 `auto-fit`，表示插件保留了 Figma 中的设计尺寸，但降低了生成 PNG 的内部像素分辨率，以避免 Figma 报 `Image is too large`。
